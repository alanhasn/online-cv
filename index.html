About Me

My name is Alan, a passionate and detail-oriented web developer with a solid foundation in Python and Kotlin. I specialize in backend development using Django, and I have a growing focus on building secure and scalable systems. My interest in cybersecurity has led me to develop several tools and applications to detect vulnerabilities and enhance protection.

I am the creator of SecureTool, a Python library published on PyPI that helps developers improve the security of their web applications through built-in scanning and analysis utilities. I’ve also developed a complete Vulnerability Scanner Web App from scratch, applying practical knowledge in Django, APIs, and web security techniques.

In the Kotlin ecosystem, I’ve completed multiple projects that demonstrate my grasp of Object-Oriented Programming and asynchronous programming using coroutines. One of them is a command-line-based Order Manager for managing orders in a simulated e-commerce store, built using Exposed ORM and clean architecture. These projects reflect my ability to combine performance, structure, and user experience even in console environments.

I’m currently improving my skills in CSS and modern UI design to build visually compelling and responsive web interfaces. I value clarity, clean code, and continuous learning. I also enjoy enhancing my English skills through technical discussions, often switching between Arabic and English to sharpen my communication.

Whether working independently or contributing in a team, I’m driven by curiosity, consistency, and a commitment to building practical tools that make a real-world impact.
